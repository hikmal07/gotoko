# GoToko
Basic E-commerce WEB App Built with GO
# Features 

- Home Page
- Register, Login, Profile User
- Products Page 
- Single Products Page
- Shopping Cart
- Checkout
    - Shipping Cost Calculation
    - Payment Gateaway
- Order History
- Admin Panel:
    - Dashboard
    - Manage Products 
    - Manage Orders 
    - Manage Users
    - Reports 
    - Configuration 
    
# Technology Stack
- Golang :
    - Gorilla Mux
    - Gorm
    - ...
- PostgreSQL
- Bootstrap
